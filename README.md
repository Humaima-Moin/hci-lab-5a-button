# 🎯 HCI Principles Demo (Interactive Web Project)

This project is a simple and interactive demonstration of key **Human-Computer Interaction (HCI)** principles using **HTML, CSS, and JavaScript**. It visually explains how good UI design improves user experience through feedback, constraints, and clear visual cues.

---

## 📌 Project Overview

The application includes:
- A **button**: "Process Data (Click me!)"
- A **status display box**

When the user clicks the button, different HCI principles are triggered and shown using colors, animations, and behavior changes.

This project is ideal for:
- HCI assignments
- UI/UX beginners
- Learning JavaScript DOM manipulation

---

## 🧠 HCI Principles Demonstrated

### 🟦 1. Affordance
- The button has a **blue border**
- Indicates that it is clickable and interactive

---

### 🟩 2. Mapping
- The **status box (green border)** shows the result of the button action
- Clearly connects user action with system response

---

### 🟨 3. Feedback
- On clicking the button:
  - Background turns **yellow**
  - Button slightly shrinks
- Confirms that the action is successfully performed

---

### 🟥 4. Constraint
- After **3 clicks**:
  - Button becomes disabled
  - Red border appears
- Prevents further interaction

---

### 🟪 5. Signifier
- Disabled button shows:
  - **Purple background**
  - Reduced opacity
- Clearly indicates that the button is no longer usable

---

## ⚙️ How It Works

1. User clicks the button
2. Counter increases
3. Status updates dynamically:
   - `Processed 1 time`
   - `Processed 2 times`
4. After 3 clicks:
   - Button is disabled
   - Status shows: `Limit reached (3 clicks max)`
